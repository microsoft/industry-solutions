## Nov-11, 2025

This release introduces initial IT system cataloging capabilities through a new model-driven app, enhanced forms and views, and dashboard functionality. The release focuses on user interface improvements and catalog management, with workflow functionality planned for future releases.

- **IT System Catalog**
    - Added new model-driven app for tracking IT systems and related components
    - Configured **Catalog** area with groups for Catalog and Admin functions
        - **Catalog group**: Dashboard page, IT Systems, and IT System Components
        - **Admin group**: IT System Component Types, IT Technologies, and IT Technology Types
    - Added **Source Systems** area with integration capabilities
        - **Integration Setup group**: Integration Identifiers, Integration Value Maps, and Integration Commit Policies
        - **Integration Processing group**: Integration Sync Jobs, Integration Sync Items, and Integration Ledgers
    - Includes dedicated IT System Catalog Dashboard for operational visibility

- **IT System**
    - Configured baseline Information form with comprehensive metadata fields
    - Added baseline Active IT Systems view for system browsing and management

- **IT System Component**
    - Removed original IT System Component Main form (replaced with Information form)
    - Added baseline Quick Create form for rapid component entry
    - Configured baseline Active IT System Components view
    - Enhanced Information form with expanded component metadata

- **IT System Catalog Dashboard**
    - Added new dashboard for system catalog overview and metrics
    - Provides operational visibility into system portfolio status
