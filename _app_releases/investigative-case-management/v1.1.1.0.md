## Jan-14, 2026

* Dependency cleanup:
  - Removed dependency on the legacy FED PCFIFrame control to improve solution portability and reduce external dependencies.

* What remains unchanged:
  - All core functionality and data models remain intact.
  - Schema and naming conventions continue to use the `msgov_` prefix established in v1.1.0.0.

* Impact and guidance:
  - This is a minor maintenance release with no functional changes to the app.
  - Agencies upgrading from v1.1.0.0 should experience no breaking changes.
