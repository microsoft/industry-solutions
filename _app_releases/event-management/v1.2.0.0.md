## November 12, 2024

This release builds upon the expanded Event Management data model v1.1.0.0 to deliver a functional model-driven app with comprehensive forms, views, and navigation for managing complex events.

### Application Structure

- **Event Management App**: Enhanced sitemap with organized areas and navigation
  - **Event Management Area**: Core event planning and execution capabilities
    - **Event Setup Group**: Events, Event Sponsors, Event Tracks, Event Sessions, Resource Reservations
    - **Event Execution Group**: Event Registrations (Event Participants), Session Registrations (Event Session Participants), Event Requests, Participant Entries (Event Entries)
    - **Organizations Group**: Organizations, Contacts, Staff
  - **Settings Area**: Administrative configuration
    - **Admin Group**: Event Types, Bookable Resources

### Enhanced Forms and Views

- **Event Information Form**: Multi-tab interface supporting the expanded event capabilities
  - **General Tab**: Basic event information, dates, and categorization
  - **Logistics Tab**: Location, resource management, and operational details
  - **Sponsors Tab**: Funding sources and sponsor management
  - **Sessions Tab**: Session planning and track organization
  - **Participants Tab**: Attendee and participant management
  - **Requests Tab**: Event-related requests and entries

- **New Entity Forms and Views**: Complete baseline configurations for all new entities
  - **Event Sponsors**: Information forms, quick create, and views for funding management
  - **Event Sessions**: Session planning with room assignments and timing
  - **Event Tracks**: Thematic organization for complex events
  - **Event Participants**: Specialized participant role management
  - **Event Session Participants**: Session-level attendance tracking
  - **Event Requests**: Request workflow management
  - **Event Entries**: Registration and entry processing

### Navigation and User Experience

- Logical grouping of related functionality for intuitive navigation
- Quick create forms for efficient data entry during event planning
- Consistent view layouts for easy information review and management
- Administrative separation for system configuration and reference data

This release transforms the Event Management module from a basic data structure into a functional application ready for event planning and execution, supporting everything from simple meetings to multi-session conferences with sponsors and detailed participant tracking.
