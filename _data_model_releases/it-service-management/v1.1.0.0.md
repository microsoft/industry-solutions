## Nov-11, 2025

This release enhances IT system and component management capabilities with expanded metadata fields, new technology categorization tables, and improved system architecture tracking. It includes significant updates to IT System and IT System Component entities, along with new reference tables for better technology management.

- **IT System**
    - Added `Description` field: New multiline rich text field, 4000 characters
    - Added `URL` field: URL field, 500 characters
    - Added `Point of Contact` lookup to Person (Contact)
    - Added `Lifecycle Stage` choice (global Lifecycle Stage)
    - Added `Operational Status` choice (global Operational Status)
    - Added `Criticality` choice (global High Medium Low)
    - Added `Catalog Status` choice (global Publication Status)
    - Added `Visibility` choice (global Visibility)
    - Added `Last Review Date` and `Next Review Date` fields: Date only
    - Added `Primary IT Hosting Model` choice (IT Hosting Model)
    - Added `Primary Platform` and `Primary Programming Language` lookups to IT Product
    - Added `Security Classification` choice (global Security Classification)
    - Added `Dependencies Summary` field: New multiline rich text field, 4000 characters
    - Added `ATO (Authority to Operate)` lookup to IT System Accreditation
    - Added `Information Security Officer` lookup to Person (Contact)
    - Added `Is Public API Available?` choice (global Yes No)
    - Added `Public API Endpoint` field: URL field, 500 characters
    - Added icon

- **IT System Component**
    - Added `IT System Component Type` lookup to IT System Component Type
    - Added `Description` field: New multiline rich text field, 4000 characters
    - Added `Tags` field: New multiline plain text field, 2000 characters
    - Added `Security Classification` choice (global Security Classification)
    - Added `Operational Status` choice (global Operational Status)
    - Added `Visibility` choice (global Visibility)
    - Added `Primary Steward` lookup to Person (Contact)
    - Added `Last Review Date` and `Next Review Date` fields: Date only
    - Added `Parent IT System Component` lookup to IT System Component
    - Added icon

- **IT Hosting Model**
    - Added new choice set with values: On-Premises, Cloud IaaS, Cloud PaaS, Cloud SaaS, Hybrid

- **IT System Component Types**
    - Added new reference table for system component categorization

- **IT Technology**
    - Added new table to replace IT Product for better technology management
    - Fields include: Name, Vendor (lookup to Account), Description

- **IT Technology Category**
    - Added new choice set with values: Platform or Infrastructure, Programming Language, Application Framework, Analytics, AI, or Model, Integration or API, Development or Deployment, Security or Compliance, Data Management / Cataloging

- **IT Technology Types**
    - Added new reference table for technology type categorization
    - Added `IT Technology Category` choice (global IT Technology Category)
    - Added `Description` and `Parent IT Technology Type` fields

- **IT Product**
    - Added icon

- **IT System Accreditation**
    - Added icon
