Getting Started

New to GLOWS?

Check out our Getting Started Guide for detailed setup instructions and best practices.

Watch the Overview

  1. Explore Personas and Use Cases

    Get familiar with common government scenarios and the roles involved via the use cases and personas areas. These give context on where the app templates and data models can help.

  2. Review Data Models and App Starter Kits

    Browse the standardized data models and app starter kits. Each includes overviews, release notes, and documentation to guide setup.

  3. Download and Deploy

    Select the latest release of the data model or app kit you need, and follow the how-to documentation to deploy in your environment.
